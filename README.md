# ecomm
# React E-Commerce Storefront Application

## Overview

This project is a responsive e-commerce storefront application built using **React** and **Redux** with automated CI/CD deployment using **GitHub Actions**. The application demonstrates modern frontend architecture, centralized state management, API integration, and pipeline-based deployment validation.

The goal of this project is to showcase end-to-end application delivery skills combining **frontend engineering** with **DevOps automation practices**.

---

## Features

* Product listing and category filtering
* Shopping cart functionality (add/remove/update items)
* Product detail views
* Login and registration pages
* Centralized state management using Redux
* REST API integration for product data
* Multi-page navigation using React Router
* Responsive UI using Bootstrap
* Automated CI/CD deployment workflow using GitHub Actions
